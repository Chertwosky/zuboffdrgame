# Фишечный поход (веб-игра)

Мини-игра с цикличным полем, броском кубика и заданиями по цветам ячеек.

## Что улучшено

- Полностью освежён визуальный стиль: более чистая типографика, прогресс до цели, улучшенные панели и контраст.
- Усилен UX игрового цикла: понятные подсказки, более читаемые статусы, аккуратная обратная связь после хода.
- Добавлен полноэкранный медиа-оверлей после результата задания:
  - при успехе/провале выбирается случайный видео- или image-контент,
  - одновременно включается случайный звук,
  - если медиафайлы ещё не добавлены, срабатывает безопасный звуковой fallback.
- Добавлен «премиум»-слой UX:
  - светлая/тёмная тема в один клик,
  - быстрый mute/unmute,
  - горячие клавиши для хода и принятия задания,
  - игровая аналитика (выполнено, успешность, серия),
  - лента событий текущей партии.
- Добавлены инструменты администрирования уровня продакшн:
  - экспорт текущего состояния в JSON,
  - импорт из JSON,
  - полный сброс конструктора и прогресса.
- Сохранены и развиты исходные идеи: цикличная карта, задания по цветам, фишка, очки, победа по очкам с финишем на старте и админ-конструктор.

## Возможности

- Цикличная карта из разноцветных непронумерованных ячеек.
- Особая стартовая ячейка.
- Бросок кубика и перемещение фишки по кругу.
- Задание открывается только после попадания на ячейку.
- На цвет ячейки можно назначить несколько заданий (выбирается случайно).
- За выполнение начисляются очки, за провал показывается штрафное задание.
- Победа: набрать целевые очки **и** вернуться на старт (финишировать).
- Скрываемая админ-панель:
  - добавление ячеек,
  - изменение цели очков,
  - добавление заданий,
  - добавление картинки и звука к заданию.

## Структура медиа-папок

Заполните эти папки своими файлами (пока по желанию):

- `media/sounds/success/` — звуки успеха
- `media/sounds/fail/` — звуки провала
- `media/images/success/` — картинки успеха
- `media/images/fail/` — картинки провала
- `media/video/success/` — видео успеха
- `media/video/fail/` — видео провала

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.
