# Фишечный поход (веб-игра)

Мини-игра с цикличным полем, броском кубика и заданиями по цветам ячеек.

## Возможности

- Цикличная карта из разноцветных ячеек с визуальным выделением текущей позиции.
- Улучшенный интерфейс: карточки статистики, прогресс до цели, более читаемые элементы управления.
- Особая стартовая ячейка.
- Бросок кубика и анимированное перемещение фишки по кругу.
- Задание открывается только после попадания на ячейку.
- На цвет ячейки можно назначить несколько заданий (выбирается случайно).
- За выполнение начисляются очки, за провал показывается штрафное задание.
- Эффекты реакции на ответ:
  - эмодзи/частицы на фоне,
  - полноэкранный медиа-оверлей,
  - при срабатывании показывается случайный **видеоролик со звуком** или **картинка со случайным звуком**.
- Победа: набрать целевые очки **и** вернуться на старт.
- Скрываемая админ-панель:
  - добавление ячеек,
  - изменение цели очков,
  - добавление заданий,
  - добавление картинки и звука к заданию.

## Папки для ассетов

Подготовлены отдельные папки, которые можно наполнять позже:

- `media/videos/success`
- `media/videos/fail`
- `media/images/success`
- `media/images/fail`
- `media/sounds/success`
- `media/sounds/fail`

Шаблон путей уже подключён в `app.js`.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.
