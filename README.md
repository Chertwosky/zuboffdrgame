# Фишечный поход (веб-игра)

Игра с цикличным полем, кубиком и заданиями по цветам ячеек.

## Что реализовано

- Цикличная карта из непронумерованных ячеек.
- Только 4 цвета: **сиреневый, розовый, зеленый, голубой**.
- У каждого цвета своя категория (Категория 1..4).
- Фишка двигается после броска кубика **по шагам через каждую промежуточную ячейку**.
- Анимация броска кубика.
- Улучшенный визуал: анимированные кнопки/карточки/поле.
- Задание открывается только после попадания на ячейку.
- Случайное задание из набора, привязанного к цвету/категории ячейки.
- За выполнение начисляются очки, за провал показывается штраф.
- Победа: набрать целевые очки и вернуться на старт.
- Скрываемая админ-панель:
  - добавление ячеек,
  - добавление/изменение/удаление заданий,
  - картинка и звук у задания,
  - настройка вида фишки (вшитые формы или своя картинка),
  - перетаскивание ячеек мышью прямо по полю.
- Эффекты результата:
  - при успехе: конфетти + случайный встроенный звуковой сигнал,
  - при штрафе: анимация 💩 + случайный встроенный звуковой сигнал.

## Запуск

```bash
python3 -m http.server 4173
```

Открыть в браузере:

`http://localhost:4173/index.html`
