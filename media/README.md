# Media assets

Заполните папки собственным контентом:

- `videos/success` — видео при правильном ответе
- `videos/fail` — видео при неправильном ответе
- `images/success` — изображения при правильном ответе
- `images/fail` — изображения при неправильном ответе
- `sounds/success` — звуки при правильном ответе
- `sounds/fail` — звуки при неправильном ответе

В `app.js` уже подготовлены шаблонные пути (`*-01`, `*-02`).
